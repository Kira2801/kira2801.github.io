Long Live Garden

Role

Game Designer

Focus

Level Design, System Design

Year

2026

Overview

Long Live Garden is a puzzle game where players must strategically move plants and obstacles to clear the path and allow the main plant to grow.

Design Challenge

The challenge was designing level progression that introduces mechanics gradually while maintaining player engagement and avoiding frustration spikes.

Design Solution

I implemented a structured level progression system using controlled mechanic introduction, difficulty ramping, and pattern variation to maintain engagement.

Design Process

- Designed level dependency graph - Created difficulty curve structure - Iterated based on playtesting - Balanced challenge and accessibility

Result

The progression system successfully improved player retention and provided a satisfying difficulty curve.
